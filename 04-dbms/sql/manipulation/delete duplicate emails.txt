solution 1:

delete from person
where id in (
    select * from (
        select p1.id from person p1 
        join person p2
        on p1.email=p2.email and p1.id>p2.id
    ) tmp
)

strategy:

we cannot delect directly from subquery, so at first we need to store the result of subquery as temporary,
then on temporary we can do delete operation.

time complexity: O(n^2)




solution 2:

delete p1 from person p1
join person p2
on p1.email=p2.email
and p1.id>p2.id

strategy:

we can use join to delete directly from table, we just need to make sure that we only delete the duplicate email with bigger id.

time complexity: O(n^2)
