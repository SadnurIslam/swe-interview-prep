Note:
-> Cross Join- Returns the Cartesian product of the two tables. (A X B)
    It combines each row from the first table with every row from the second table. 
    If there are no matching records in either table, the result will still include all combinations of rows from both tables.

-> Ei problem a count(*) dile vul hobe, count(*) dile null gulao 1 count korbe kintu null gulo 0 hobe ans,
    tai count(e.student_id) dile null gulo count hobe na, karon e.student_id null hole count hobe na.
    tai count(e.student_id) othoba count(e.subject_name) dite hbe


    
Solution:

select st.student_id, st.student_name, s.subject_name, count(e.student_id) as attended_exams
from Students st
cross join Subjects s
left join Examinations e
on st.student_id=e.student_id and s.subject_name=e.subject_name
group by st.student_id, s.subject_name
order by student_id, subject_name asc
