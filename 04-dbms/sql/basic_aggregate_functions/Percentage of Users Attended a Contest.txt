Solution (Optimized, Complexity Low):

select contest_id, 
round((select count(user_id)/(select count(*) from Users))*100,2)
as percentage
from Register
group by contest_id
order by percentage desc, contest_id asc




Solution:  (Brute Force, Complexity High)

select r.contest_id, 
round(avg(
    case
        when r1.user_id is null then 0
        else 1
    end
)*100,2) as percentage
from Register r
cross join Users u
left join Register r1
on r.contest_id=r1.contest_id and u.user_id=r1.user_id
group by r.contest_id
order by percentage desc, contest_id asc
